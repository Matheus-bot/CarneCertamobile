# 🥩 CarneCerta Mobile

Aplicativo Android desenvolvido em AndroidStudio-Java inspirado no projeto CarneCerta.

## Funcionalidades atuais

-> Navegação entre telas
-> Integração com SQLite
-> Banco de dados local ROOM
-> CRUD simples de carnes
-> Interface Light and Dark Mode

-> Firebase Authentication  
-> Cadastro de usuários  
-> Login com email e senha  
-> Persistência local de dados  
-> Interface mobile responsiva  
-> Login com google 
-> Estrutura da geolocalização 

## Tecnologias utilizadas

-> Java
-> Android Studio
-> SQLite
-> XML
-> Git & GitHub

## Objetivo

Projeto acadêmico desenvolvido para aprendizado de:
-> desenvolvimento mobile
-> banco de dados
-> Firebase
-> geolocalização
-> arquitetura básica Android
