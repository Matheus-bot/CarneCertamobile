# 🥩 CarneCerta Mobile

Aplicativo Android desenvolvido em AndroidStudio-Java inspirado no projeto CarneCerta.

## Funcionalidades atuais

-> Navegação entre telas
-> Integração com SQLite
-> CRUD simples de carnes
-> Interface Dark Mode
-> Estrutura inicial Android Studio

## Tecnologias utilizadas

-> Java
-> Android Studio
-> SQLite
-> XML
-> Git & GitHub

## Objetivo

Projeto acadêmico desenvolvido para aprendizado de:
-> desenvolvimento mobile
-> banco de dados
-> Firebase
-> geolocalização
-> arquitetura básica Android
